# 🚀 Sistema de Rotina Inteligente

Projeto desenvolvido em *C# Console* utilizando os principais conceitos de *Programação Orientada a Objetos (POO)*.

## 📋 Objetivo

Automatizar tarefas comuns do computador através de um sistema de rotinas executadas por um menu interativo.

---

# 🛠️ Tecnologias Utilizadas

- C#
- .NET
- Programação Orientada a Objetos (POO)
- Manipulação de Arquivos e Diretórios
- Console Application

---

# 📚 Conceitos Aplicados

✅ Interface

✅ Classe Abstrata

✅ Encapsulamento

✅ Herança

✅ Polimorfismo

✅ Listas (List<>)

✅ Tratamento de Exceções (try/catch)

✅ Métodos Privados

✅ Organização em Camadas
